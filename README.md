# Google Maps Business Data Collection

## Overview

This repository showcases a sample dataset collected using a custom Python scraper. The extracted data is intended for portfolio demonstration purposes and highlights the type of business information that can be collected from Google Maps.

> **Note:** This repository contains a sample dataset only. The scraper source code is not included.

---

## Information Collected

The dataset includes the following fields:

- Source URL
- Business URL
- Business Name
- Description
- Category
- Phone Number
- Website
- Email Address
- Email Type
- Full Address
- Street
- City
- Region / State
- Country Code
- Language
- Plus Code
- Latitude
- Longitude
- CID
- Google Place ID
- Services
- Profile Photo
- Photo Count
- Photo URLs
- Reviews URL
- Average Rating
- Total Reviews
- Business Status
- Opening Hours (Monday–Sunday)
- Scraped Date
